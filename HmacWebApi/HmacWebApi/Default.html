Test Links

The following links are for testing and viewing responses from this website and the PowerShell backend. Hitting F12 in your browser and viewing the Network tab is a handy way to view the response data.

To view API details, see the project wiki at https://github.iu.edu/UITS-CCI-Test/Demo/wiki.

REST Service (no call to PowerShell backend)

These responses reflect situations happening prior to calling PowerShell.

  • Value returned (http 200 OK)
  • Action method exception (http 400 Bad Request)
  • Constructor exception (http 404 Not Found)
  • Content write exception (unhandled - http 500 Internal Server Error)
  • 500 (handled - http 500 Internal Server Error)
  • GeneralServerError from Strings.json (unhandled - writes to API EventLog)
  • AccountAlreadyEnabled from Strings.json using formatted string (unhandled - writes to API EventLog)
